# Wavelet Crash Analysis Notebooks

This repository is prepared for Binder deployment.

## Run on Binder

Use your Binder URL with this launch target:

`?urlpath=lab/tree/index.ipynb`

This opens `index.ipynb`, which links to all analysis notebooks.

## Included notebooks

- `index.ipynb` (landing page)
- `wavelet-crash-exploration.ipynb`
- `wavelet-crash-exploration_2.ipynb`
- `wavelet-crash-spatio-temporal_grid-map-overlay.ipynb`
- `wavelet_coi.ipynb`

## Environment

Binder dependencies are defined in:

- `requirements.txt`
- `runtime.txt`
